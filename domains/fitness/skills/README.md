# Fitness Domain Skills

Place fitness-domain skills here when they are reusable inside the fitness domain but not yet suitable for `skills/base`.

Examples:

- fitness training-loop planning;
- fitness product release readiness;
- fitness-domain monitoring workflows;
- project memory projections from `aifitness`.

If a skill becomes useful outside fitness, extract the generic part into `skills/base` and leave only domain vocabulary or examples here.
